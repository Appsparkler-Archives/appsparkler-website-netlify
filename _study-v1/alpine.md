### APK
- INSTALL software - `apk add <softwareName>` - for ex. `apk add git`

### CHMOD
- CHANGE permission - `chmod <permissionID> <path/to/file>`

### STAT
- GET stat of a file - `stat -c %a <path/to/file>`

### HOW TOs?
  - INSTALL `ssh`?
    - UPDATE `apk` - `apk update`
    - ADD `openssh` - `apk add openssh`
